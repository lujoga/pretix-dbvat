{% trans "Why have I received two eCoupons?" %}

{% blocktrans trimmed %} In order to allow for greater flexibility - for example when booking the outward and return journey separately - you have been provided with two eCoupon codes as every code can only be used once. You do however not need to redeem both eCoupons: it is up to you to redeem both, one or none. In case you are already set on your outward and return journey, you can purchase them both at the same time using only one eCoupon. {% endblocktrans %}

{% trans "Does the DB Event Discount/Event Ticket include my entrance ticket to the event?" %}

{% blocktrans trimmed %} No. The DB Event Discount (and the former DB Event Ticket) are an offer which allows you to purchase discounted long distance train tickets. However, these tickets do not include the entrance ticket to the event itself. As the same, purchasing an event ticket does not include a train ticket for travel from/to the event venue. {% endblocktrans %}
